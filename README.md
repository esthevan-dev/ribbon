## Why Ribbon exists

Tools should not get in the way of thought. Ribbon behaves like a continuous typewriter ribbon, where visual noise is reduced to zero and the interface only appears when you actually need it. The rest of the time, there is only you and the blank page.

Built entirely with HTML, CSS, and vanilla JavaScript. No frameworks, no heavy dependencies, no promises broken by a third party update. Just the web, in its most honest form, packaged as a native desktop app.

<br>

## Core principles

### Total focus
No separate editing and preview modes. What you type is what you see, instantly, with zero friction between thought and screen.

### Native privacy
Your notes live only on your own machine. Nothing is uploaded, synced, or tracked. We use IndexedDB to store everything locally, because your drafts belong to you alone.

### Pure and light
No external frameworks. No dependency that will eventually stop receiving updates and break your flow. Just fundamental, stable, future proof technology.

### Quiet organization
A lightweight tag system and instant search, designed to help you find an old idea without ever losing focus on the one you are having right now.

### Peace of mind
Autosave and built in version history. Write without fear. Nothing gets lost here.

## Development

Ribbon uses Electron to package the web app into a native desktop application. To run the project from source:

```bash
git clone https://github.com/esthevan/ribbon.git
cd ribbon
npm install
npm start
```

To build your own executable:

```bash
npm run build
```
